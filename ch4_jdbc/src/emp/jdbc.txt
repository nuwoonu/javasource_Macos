JDBC 프로그래밍 순서
1. 드라이버 로드
Class.forName("oracle.jdbc.OracleDriver"); // 오라클
Class.forName("com.mysql.cj.jdbc.Driver"); // MySQL

2. Connection 얻기
(1) 오라클 CASE 
url, user, password 정보
String url = "jdbc:oracle:thin:@localhost:1521:xe";
String user = "사용자 아이디";
String password = "비밀번호";

(2) MySQL CASE
String url = "jdbc:mysql//localhost:3306/database명";
String user = "사용자아이디";
String password = "비밀번호";

Connection con = DriverManager.getConnection(url,user,password);

3. SQL 문
(1) Connection을 통해서 실행할 SQL 구문 전송
    con.prepareStatement("실행할 SQL 구문");
    순서
    1. SQL 구문을 컴파일(구문 분석)
    2. 실행계획 세우기
(2) SQL 구문 실행
    1. select문 이라면 executeQuery();
    2. update,delete,insert 라면 executeUpdate();

(3) 실행결과 받기
 1. select 라면 ResultSet 에 담겨져 옴
 2. update, delete, insert 라면 int 값으로 넘어온다.
